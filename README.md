QuestEd is a gamified learning platform that enables users to learn new skills by transforming the learning process into an interactive quest-based experience. Each skill is structured as a series of quests, challenges, and milestones, motivating learners through experience points (XP), levels, streaks, and achievements.

---

##  Motivation

Traditional self-learning platforms often suffer from low engagement and a lack of structure. Learners struggle with consistency, motivation, and clear progression paths.

QuestEd addresses this problem by:
- Converting learning paths into **quests**
- Rewarding progress through **gamification mechanics**
- Encouraging consistency using **streaks and milestones**

---

##  Key Features (MVP)

- User authentication (signup/login)
- Interest and skill selection
- Quest-based learning paths
- XP and level system
- Learning streak tracking
- Gamified user dashboard
- Progress and milestone tracking
---

##  Gamification Elements

- **Quests:** Structured learning units for each skill  
- **XP (Experience Points):** Earned on completing quests  
- **Levels:** Progression based on XP  
- **Streaks:** Reward consistency in learning  
- **Badges:** Achievements for milestones (planned)
