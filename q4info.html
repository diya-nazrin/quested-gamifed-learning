Quest 4: The Spell Forge

What is a Function?

A function is like a spell you can cast again and again. You define it once, then reuse it anytime.

The Function Spell Format

In Python, functions are created using def:

def cast_spell(mana, level):
  result = mana * level
  return result

Key Parts (Very Important)

  • def → starts the function spell
  • mana, level → inputs (parameters)
  • return → sends the answer back

Your Mission: In the Spell Forge, complete the function logic and test it on crystals. If your formula is correct, the crystals will glow with the right power!
